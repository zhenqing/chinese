contMenu object
Send Feedback Back to top

This section desribes contMenu object, its properties and child objects. The contMenu is a child object of imagesList. It represents an available by right click context menu.

Structure of section:

  • Properties of contMenu object

  • Objects of contMenu object


List of properties

Property name Type Description Default value
hideBuiltInItems Boolean Indicates whether flash built-in menu items (except Settings) are displayed in the context menu. true
hideSortItems Boolean Indicates whether sorting menu items are displayed in the context menu. true
Examples:

The property of the contMenu is assigned via flashvars object of SWFObject on the html page: 

flashvars["view.imagesList.contMenu.hideBuiltInItems"] = true;

The contMenu has the following menuItem objects:


addItem - shows a "Browse file" dialog.

List of properties:

Property name Type Description Default value
visible Boolean Indicates whether menu item is displayed in the context menu. true
enabled Boolean Indicates whether menu item is enabled. true
caption String Gets or sets name of menu item that is displayed to the user. "Add files"
Examples:

The property of the addItem is assigned via flashvars object of SWFObject on the html page: 

flashvars["view.imagesList.contMenu.addItem.visible"] = true;



removeItem - removes all selected files in the images list.

List of properties:
