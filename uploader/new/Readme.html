EasyAlgo EAFlashUpload v 2.4

EAFlashUpload is a reliable, client-side flash file upload control, which can be used for solving one of the most common tasks for webmasters – cross-browser multiple file upload with high level control of the upload process. The EAFlashUpload tool provides expansive functionality for customization and it can be easy integrated into your website design. JavaScript API provides full control of user interaction with EAFlashUpload. Scalable architecture allows loading only necessary content on the client-side. The EAFlashUpload runs on the client side and is compatible with all server side platforms (PHP, ASP.NET, ASP, ColdFusion, JSP and Perl).
