This example demonstrate how to place EAFlashUpload inside collapsible panel.

Before running this example set an appropriate URL for uploadUrl property in the code of current html page (for ASP.NET: "ServerScripts/ASP.NET/uploadwiththumbnails.aspx"; or for PHP: ServerScripts/PHP/uploadwiththumbnails.php).

Note: Trial version has two limitiations: maximum files count is 10 and size of each file cannot be larger than 50 MB.
