EAFlashUpload v 2.4 examples.

This package includes various examples which demonstrate how to use EAFlashUpload component to upload multiple files to the web server.

All client-side code is reperesented as set of html page, in real application you can place EAFlashUpload inside any page (.aspx, php, asp etc.). The ServerScripts folder of examples package contains ASP.NET, PHP, ColdFusion, Perl, JSP scripts for receiving files.

Before running any of below examples set an appropriate URL for uploadUrl property as described in appropriate example. If you are interested in ASP.NET examples ensure that ASP.NET Application is started for "ServerScripts/ASP.NET" virtual directory.

Note: Trial version has two limitiations: maximum files count is 10 and size of each file cannot be larger than 50 MB.

Basic examples:

Simple upload Demonstrates how to upload multiple files to the web server.
Uploader inside collapsible panel Demonstrates how to place EAFlashUpload into collapsible panel.
Add files description and parse form values Demonstrates how to add and send specific description for eash file.
Create a custom view for files list. Demonstrates how to add custom columns to the files list and include custom form elements(checkbox, radio button) to cells.
JavaScript user interface Demonstrates how to create "pure" JavaScript/DHTML interface and upload multiple files to the server.
Change properties at the runtime Displays available user interfaces. Demonstrates how to change properties of EAFlashUpload at the runtime.

Images processing examples:

Upload multiple images and thumbnails creation Demonstrates how to upload multiple images to the web server. Generate and display thumbnails of uploaded images.
Simplest image gallery example Demonstrates how to upload multiple image files to the server and organize them as simple gallery.
Multiple images upload with preview them on client-side. Demonstrates how to upload multiple images to the web server. The preview thumnails are generated for each image.
Upload multiple images with resizing on the client-side Demonstrates how to resize multiple images on the client-side and uploading them to the server.
